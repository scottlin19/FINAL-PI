Programa para el procesamiento de datos de datos de vuelos desde y hacia aeropuertos nacionales.

Como generar ejecutable?
Escribiendo el comando make dentro de la misma carpeta generará el ejecutable llamado "censo".

Como ejecutar el programa?
Para ejecutar el programa basta con escribir el comando "./censo pathAP pathMOV" con pathAP siendo el path hacia el archivo "aeropuertos.csv" y pathMOV siendo el path hacia el archivo "movimientos.csv". 
ATENCIÓN: inversar los paths no generará los resultados esperados.
